# agent-hype

agent finishes task.
terminal throws party.

works with Claude Code

## Install

```bash
npx skills add [User]/agent-hype
claude plugin install agent-hype@[abdelrahmanmagdii]
```

need `python3`.
need `rich`.

```bash
python3 -m pip install rich
```

## Usage

Triggered automatically when task done. No config. Just Celebrations.


## Files

- `.claude-plugin/hooks.json` wires Claude Code to the celebration hook
- `skills/hype/SKILL.md` teaches Cursor and Codex the same victory move
- `scripts/hype.py` does the actual hype
